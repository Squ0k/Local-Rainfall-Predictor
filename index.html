This application predicts whether rainfall will occur within the next 2 hours. To run a prediction, first upload or capture an image of the sky.

The current model achieves an accuracy of approximately 80%. For optimal results, use images of the sky taken during daytime, and avoid including obstructions such as trees, powerlines, or buildings.

The source code for this project is available on Github: https://github.com/Squ0k/Local-Rainfall-Predictor. For inquiries or feedback, please contact the developer at kt.92.12021@gmail.com.
